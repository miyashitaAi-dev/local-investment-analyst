# 保有中ポジション中間チェックスキル

## 事前確認

ユーザーがエントリー時の情報（エントリー価格・当時の判断根拠）を伝えていない場合は、検索を始める前に一度だけ確認の質問をしてよい（これはコアルール第1条の「前置き禁止」の例外）。情報が得られたら、それ以降は前置きなしでレポート本文のみ出力する。

## 検索ワークフロー（レポート作成前に`search_web`で必ず全て実行する）

1. `[対象] 現在価格 [現在の年]`（`[現在の年]`は、コアルールで最初に検索確認した「今日の日付」の年を使う）
2. `[対象] 最新ニュース`
3. 対象がFX・暗号資産・コモディティの場合は、`fx-crypto-commodity-analysis`スキルのグループ判定（グループA/B）に従ってテクニカル評価方法を選ぶ
4. 必要に応じて`[対象] RSI MACD 移動平均`も検索し、エントリー時との変化を確認する

検索結果が薄い項目は最大2回まで再検索する。それでも見つからなければ`N/A`とする（コアルール第0条に従う）。
